\section{技术栈说明}

前端基于 Vue 3 Composition API + TypeScript 架构，使用 Vite 5 构建，Element Plus 作为 UI 组件库，Pinia 状态管理，Axios 封装 HTTP 请求。管理后台统计图表使用 ECharts 5 实现数据可视化。

\section{详细工作内容}

\subsection{第一周（7月4日—7月5日）：首页与商品模块}

\subsubsection*{首页开发}
\begin{itemize}
    \item 实现 Hero 横幅：渐变绿色背景 + 浮动几何圆形装饰动画 + 品牌标语
    \item 实现实时统计数据展示：注册用户数、在售商品数、成交订单数（对接 /index/statistics 接口）
    \item 实现搜索框：圆角输入框 + 橙色搜索按钮，支持回车搜索和点击搜索，跳转商品列表页并携带关键词参数
    \item 实现公告栏：仿真实公告板设计（绿色胶带装饰条），公告标题水平滚动轮播（CSS Animation），鼠标悬停暂停
    \item 实现商品分类导航：8 个分类 Tab 按钮（含 Emoji 图标），点击切换加载对应分类商品，选中态绿色高亮
    \item 实现首页商品网格：响应式 Grid 布局（240px 最小列宽），卡片含封面图（hover 放大效果）、分类标签、标题、价格、浏览量和时间
    \item 实现相对时间格式化函数：刚刚 / X分钟前 / X小时前 / X天前 / 日期
\end{itemize}

\subsubsection*{商品列表页开发}
\begin{itemize}
    \item 实现多条件筛选栏：关键词搜索、分类下拉、价格区间、商品成色、排序方式（价格/时间/浏览量）
    \item 对接后端分页 API，支持 URL query 参数同步（刷新保持筛选状态）
    \item 实现分页组件（Element Plus Pagination），每页 20 条，支持页码和每页条数切换
    \item 实现空状态提示：无结果时展示 El-Empty 组件
\end{itemize}

\subsubsection*{商品详情页开发}
\begin{itemize}
    \item 实现图片画廊：主图大图展示 + 缩略图列表，点击缩略图切换主图

    \item 实现操作按钮组：加入购物车（带图标）、立即购买（双行文字）、收藏（Star/StarFilled 图标切换）

    \item 实现响应式布局：移动端单列显示，主图高度自适应

\subsubsection*{商品发布页开发}

    \item 每件商品包含真实描述、合理价格和原价、正确成色、随机浏览量和收藏数

\subsubsection*{图片方案优化}
\begin{itemize}
    \item 参与三版图片方案迭代：LoremFlickr（不稳定）→ Picsum.photos 随机（无主题）→ 按分类种子前缀
    \item 最终方案：20 种分类种子前缀（phone-/laptop-/textbook-/shoes-/sport- 等），同分类商品共享主题，不同 ID 保证差异化
    \item 用户头像方案：迁移到 Pravatar（基于用户名种子稳定生成）
\end{itemize}

\section{核心技术亮点}

\begin{enumerate}
    \item \textbf{级联分类选择器}：使用 Element Plus Cascader 实现二级分类选择，父分类可展开查看子分类，选择体验流畅。
    \item \textbf{ECharts 可视化}：饼图展示分类占比、折线图展示时间趋势，图表自适应窗口大小变化，组件卸载时释放资源防止内存泄漏。
    \item \textbf{分类占位图回退机制}：商品封面图加载失败时根据分类名称返回对应 Emoji 占位封面，比纯灰色占位图更友好。
    \item \textbf{HTML 内容安全净化}：商品描述使用正则移除 script/style 标签后剥离所有 HTML 标签，再通过 DOM textarea 解码 HTML 实体，三重防线防止 XSS 攻击。
    \item \textbf{全功能管理后台}：涵盖用户管理、商品审核、订单管理、数据统计、公告管理五大模块，支持搜索、筛选、分页、编辑、状态流转全流程操作。
    \item \textbf{响应式设计}：所有页面适配移动端（小于等于768px），商品网格从多列自适应为双列，导航栏和侧边栏在小屏下自动隐藏或折叠。
\end{enumerate}
