## 1. 平台简介

**转转**是一款面向高校学生的校园二手物品交易平台，旨在为在校师生提供一个安全、便捷、可靠的二手物品买卖渠道。

### 1.1 平台特色

- **校园专属**：仅面向校内师生，交易环境纯净
- **安全可靠**：实名认证 + 信誉体系，保障交易安全
- **便捷高效**：支持商品搜索、分类筛选、站内信沟通
- **全程追踪**：订单状态实时更新，交易流程透明可控

### 1.2 主要功能

| 功能模块 | 说明 |
|:---------|:------|
| 商品浏览 | 分类浏览、关键词搜索、价格筛选 |
| 商品发布 | 上传图片、填写描述、自主定价 |
| 购物车 | 暂存心仪商品、批量管理 |
| 订单管理 | 下单、支付、发货、收货、评价 |
| 站内信 | 与卖家/买家实时沟通 |
| 个人中心 | 信息管理、地址管理、收藏管理 |

## 2. 快速入门

### 2.1 访问平台

打开浏览器，在地址栏输入平台网址即可进入转转首页。

**推荐浏览器**：Chrome 80+ / Firefox 80+ / Edge 80+ / Safari 14+

### 2.2 首页概览

首页自上而下包含以下区域：

1. **顶部导航栏**：Logo、菜单导航、登录/注册入口
2. **搜索栏**：输入关键词搜索商品
3. **数据统计**：平台用户数、在售商品数、成交订单数
4. **商品分类**：按类别展示商品入口
5. **推荐商品**：精选推荐商品列表
6. **页脚**：使用指南、联系我们

## 3. 用户注册与登录

### 3.1 注册账号

1. 点击首页右上角的「注册」按钮
2. 选择注册方式：
   - **邮箱注册**：填写用户名、邮箱地址、密码，获取邮箱验证码
   - **手机注册**：填写用户名、手机号、密码，获取短信验证码
3. 填写收到的 6 位验证码
4. 阅读并同意用户协议
5. 点击「注册」按钮完成注册

**密码要求**：
- 长度 8-20 位
- 必须包含字母和数字
- 建议使用大小写字母 + 数字组合

### 3.2 登录账号

1. 点击首页右上角的「登录」按钮
2. 输入账号（用户名 / 邮箱 / 手机号）
3. 输入密码
4. 点击「登录」按钮

### 3.3 找回密码

1. 在登录页面点击「忘记密码」
2. 输入注册时使用的邮箱或手机号
3. 获取验证码
4. 输入新密码并确认
5. 点击「重置密码」完成

## 4. 个人中心

### 4.1 编辑个人资料

登录后点击右上角头像 →「个人中心」，可编辑以下信息：

| 项目 | 说明 |
|:----|:------|
| 头像 | 点击头像上传，支持 JPG/PNG/GIF/WebP 格式 |
| 昵称 | 给自己取一个个性化昵称（最多 20 字） |
| 简介 | 简单介绍自己（最多 200 字） |
| 性别 | 保密/男/女 |
| 手机号 | 绑定手机号用于账号安全 |

### 4.2 收货地址管理

在个人中心侧边栏点击「收货地址」：

1. **新增地址**：填写收件人、手机号、所在地区、详细地址
2. **设为默认**：设置常用地址为默认收货地址
3. **编辑地址**：修改已保存的地址信息
4. **删除地址**：移除不再使用的地址

> 首次添加的地址会自动设为默认地址。

### 4.3 我的收藏

在商品详情页点击「收藏」按钮，商品会保存到「我的收藏」中。可在个人中心查看和管理收藏的商品。

### 4.4 我的订单

在个人中心可查看所有订单，按角色分为「买家订单」和「卖家订单」。

## 5. 浏览与搜索商品

### 5.1 分类浏览

在首页商品分类区域，点击感兴趣的分类图标，即可查看该分类下的全部商品。

商品分类包括：
- 电子产品
- 书籍教材
- 生活用品
- 服饰鞋包
- 运动户外
- 其他

### 5.2 关键词搜索

在首页搜索框中输入关键词，按回车或点击「搜索」按钮，系统将搜索商品标题和描述中包含该关键词的商品。

### 5.3 高级筛选

在商品列表页，可以通过以下条件组合筛选：

| 筛选条件 | 说明 |
|:---------|:------|
| 分类 | 按一级分类筛选 |
| 价格区间 | 设置最低价和最高价 |
| 成色 | 全新 / 几乎全新 / 轻微使用痕迹 / 明显使用痕迹 / 老旧 |
| 排序 | 按发布时间、价格、浏览量、收藏数排序 |

### 5.4 商品详情

点击商品卡片进入详情页，可查看：

- 商品图片
- 商品标题与描述
- 价格与成色
- 卖家信息（昵称、信誉分）
- 发布时间与浏览次数
- 收藏与分享按钮

## 6. 发布商品

### 6.1 发布流程

1. 登录后点击导航栏的「发布」按钮
2. 填写商品信息：

| 字段 | 说明 | 必填 |
|:----|:------|:----:|
| 商品标题 | 5-50 字，简明扼要描述商品 | ✅ |
| 商品描述 | 详细介绍商品情况、购买时间、使用感受等 | ❌ |
| 商品价格 | 设置售价（元） | ✅ |
| 原价 | 商品的购买原价 | ❌ |
| 商品成色 | 选择商品的新旧程度 | ✅ |
| 商品分类 | 选择正确的分类 | ✅ |
| 商品图片 | 上传商品实物照片 | ❌ |

3. 点击「发布」按钮提交审核

### 6.2 发布规范

- 商品标题应真实准确，不得含有虚假信息
- 商品描述应详细说明商品状况
- 图片应为商品实物照片，不得使用网络盗图
- 禁止发布违禁品、虚拟商品

### 6.3 审核流程

商品发布后进入「待审核」状态，管理员审核通过后才会在商品列表中展示。一般情况下：

- 审核时间：1-2 小时
- 审核未通过时将收到通知说明原因

## 7. 购物车

### 7.1 添加商品到购物车

在商品详情页点击「加入购物车」按钮，选择数量后即可将商品加入购物车。

### 7.2 管理购物车

在购物车页面可以：

- 修改商品数量
- 选中/取消选中商品
- 删除购物车中的商品
- 查看商品价格小计

### 7.3 从购物车下单

1. 勾选需要购买的商品
2. 点击「去结算」按钮
3. 选择收货地址
4. 提交订单

---

## 8. 下单与交易

### 8.1 下单流程

1. 在商品详情页点击「立即购买」或从购物车结算
2. 选择收货地址
3. 填写备注（可选）
4. 提交订单

### 8.2 订单状态

| 状态 | 说明 | 操作方 |
|:----|:------|:------:|
| 待付款 | 订单已创建，等待买家付款 | 买家 |
| 待发货 | 买家已付款，等待卖家发货 | 卖家 |
| 待收货 | 卖家已发货，等待买家确认收货 | 买家 |
| 已完成 | 买家已确认收货，交易完成 | 系统 |
| 已取消 | 订单已被取消 | 买家/卖家 |

### 8.3 买家操作

1. **付款**：在「待付款」状态下点击「去付款」完成支付
2. **确认收货**：收到商品后，在「待收货」状态下点击「确认收货」
3. **取消订单**：在「待付款」状态下可取消订单
4. **评价**：交易完成后可对卖家进行评价（1-5 星）

### 8.4 卖家操作

1. **发货**：收到订单后，在「待发货」状态下填写物流信息并发货
2. **查看订单**：在「我的订单」中查看买家信息

### 8.5 交易注意事项

- 建议选择校内当面交易，核实物品状况后再确认收货
- 保留交易过程中的聊天记录和凭证
- 如遇纠纷，可联系平台客服介入处理

## 10. 常见问题

### 10.1 账号问题

**Q：忘记密码怎么办？**
A：在登录页面点击「忘记密码」，通过注册邮箱或手机号验证后即可重置密码。

**Q：如何修改绑定的手机号？**
A：在「个人中心」→ 编辑资料中，可绑定或更换手机号。

**Q：账号被禁用了怎么办？**
A：请联系平台客服了解禁用原因并申请解封。

### 10.2 交易问题

**Q：如何联系卖家？**
A：在商品详情页点击「联系卖家」按钮，通过站内信与卖家沟通。

**Q：付款后卖家不发货怎么办？**
A：在订单详情页点击「申请退款」，平台将介入处理。

**Q：收到的商品与描述不符怎么办？**
A：在订单详情页申请退款或联系卖家协商，平台客服可提供协助。

### 10.3 商品问题

**Q：商品发布后多久审核通过？**
A：一般情况下 1-2 小时内完成审核，请耐心等待。

**Q：如何修改已发布的商品信息？**
A：在商品详情页点击「编辑」按钮即可修改商品信息。

**Q：如何下架商品？**
A：在商品详情页将商品状态改为「已下架」即可。

### 10.4 技术问题

**Q：页面加载异常怎么办？**
A：尝试刷新页面或清除浏览器缓存后重试。

**Q：图片上传失败怎么办？**
A：请检查图片格式（支持 JPG/PNG/GIF/WebP）和大小（不超过 5MB）。
