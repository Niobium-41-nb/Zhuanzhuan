## III. Color Scheme
- Primary: #2D6A4F (forest green)
- Secondary: #52B788 (light green)
- Accent: #E76F51 (warm orange)
- Background: #FDFBF7 (warm white)
- Surface: #FFFFFF
- Text: #1F2937 (dark gray)
- Muted: #6B7280 (medium gray)
- Border: #E5E7EB

## IV. Typography
- CJK heading: Noto Sans SC Bold
- CJK body: Noto Sans SC Regular
- Latin: system sans-serif
- Body size: 18px
- Heading sizes: 36px (H1), 24px (H2), 20px (H3)

## V. Brand Element
- Top-right: HZAU logo (hzau-logo.png), width 120px, positioned at x=1120, y=12
- Bottom bar: 3px primary color line at y=714
- Page number: bottom-right, 12px muted

## VI. Layout Pattern
- Cover: centered title + project name + team members + date
- Content pages: 80px left margin, title at y=60, content grid below
- Person pages: name + role header, 2-column content grid
- Code/tech pages: left text + right diagram
- End page: centered "Thank You"
